*	모든 테이블은 고유 식별자(ID)를 보유
*	3 Layer Architecture 에 따라 각 Layer의 목적에 맞게 개발
*	CRUD 필수 기능은 모두 데이터베이스 연결 및 JPA를 사용해서 개발
*	JDBC와 Spring Security는 사용x
*	인증/인가 절차는 JWT를 활용하여 개발
* JPA의 연관관계는 양방향으로 구현
